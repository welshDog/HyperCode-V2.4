```bash

  -H "Content-Type: application/json" \

      "type": "create_api",
      "description": "Create a simple GET /api/hello endpoint that returns {\"message\": \"Hello from HyperCode!\", \"timestamp\": <current_time>}",

```

## 🐛 **If Something Goes Wrong**
### **Issue: No approval appears on dashboard**
```bash
# Check WebSocket connection
docker-compose logs crew-orchestrator | grep websocket

# Test WebSocket manually
wscat -c ws://localhost:8081/ws/approvals
```

**Fix**: Make sure WebSocket endpoint is enabled in orchestrator.

***
### **Issue: Agent returns error**
```bash
# Check agent logs
docker-compose logs backend-specialist
docker-compose logs frontend-specialist

# Common errors:
# - "No RAG results" → Bible not ingested
# - "Import error" → Missing dependency
# - "Redis connection failed" → Redis not running
```

***
### **Issue: RAG returns no results**
```bash
# Re-ingest the Bible
docker-compose exec crew-orchestrator python -c "
from shared.rag_memory import AgentMemory
memory = AgentMemory('backend-specialist')
memory.ingest_document('/app/HYPER-AGENT-BIBLE.md')
print(f'Chunks stored: {memory.collection.count()}')
"
```
