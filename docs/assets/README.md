## Documentation Assets

This folder is the recommended home for documentation assets that should be versioned alongside the docs (diagrams, small images, example payloads).

### Screenshots

If you add screenshots to the repository:
- Put them under `docs/assets/screenshots/`
- Use stable, descriptive filenames (avoid spaces)
- Keep file sizes reasonable and prefer PNG

If screenshots are stored outside git (recommended for large sets), keep `docs/screenshots-gallery.md` accurate and store images in a shared team location.
